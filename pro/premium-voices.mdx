## Voice settings

All Pro voices support the following adjustments in TTS Voice Wizard:

- **Pitch** — raise or lower the voice pitch using the pitch slider
- **Speaking rate** — speed up or slow down delivery using the rate slider
- **Volume** — adjust output volume independently of system volume
- **Style** — select an expressive style where supported (Azure and IBM Watson Expressive voices)

<Note>
  The exact number and availability of voices depends on each provider — voice lists are updated by the providers periodically. TTS Voice Wizard loads voice lists from bundled JSON files that are updated with new releases.
</Note>

<Tip>
  Azure voices often have the most style options and best naturalness for VRChat use — start there if you're unsure which provider to pick.
</Tip>
