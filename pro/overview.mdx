Subscribe once and everything is handled through TTS Voice Wizard's backend — no Azure portals, no AWS consoles, no IBM dashboards.

## Pricing

Subscribe through either platform — both grant the same VoiceWizardPro benefits:

- [Ko-Fi tiers](https://ko-fi.com/ttsvoicewizard/tiers)
- [Patreon](https://www.patreon.com/ttsvoicewizard)

After subscribing, you'll receive a key that you enter directly in TTS Voice Wizard to activate your subscription.

<Note>
  VoiceWizardPro is optional — TTS Voice Wizard is fully functional as a free app with System Speech TTS and Windows built-in STT. You only need a Pro key to access cloud voices, Deepgram Nova-2, and managed translation.
</Note>
